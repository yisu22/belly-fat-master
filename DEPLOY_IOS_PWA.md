# 뱃살 마스터 iPhone 설치 배포 가이드

## 1. GitHub에 올리기
1. GitHub에서 새 저장소를 하나 만듭니다.
2. 이 폴더 전체를 그 저장소에 업로드합니다.
3. 기본 브랜치 이름을 `main` 으로 둡니다.

## 2. GitHub Pages 켜기
1. 저장소의 `Settings` 로 이동합니다.
2. `Pages` 메뉴를 엽니다.
3. `Build and deployment` 에서 `Source` 를 `GitHub Actions` 로 선택합니다.
4. 잠시 기다리면 `Deploy To GitHub Pages` 워크플로가 실행됩니다.

## 3. 공개 주소 확인
1. 배포가 끝나면 GitHub Pages 주소가 생성됩니다.
2. 보통 주소는 `https://사용자명.github.io/저장소명/` 형태입니다.
3. 이 주소를 아이폰 Safari에서 엽니다.

## 4. iPhone 홈 화면에 설치
1. Safari에서 배포 주소를 엽니다.
2. 하단 `공유` 버튼을 누릅니다.
3. `홈 화면에 추가` 를 누릅니다.
4. `Open as Web App` 를 켭니다.
5. `추가` 를 누릅니다.

## 5. 오프라인 사용 팁
1. 설치 후 앱을 한 번 이상 완전히 열어 둡니다.
2. 주요 화면을 몇 번 이동해 앱 리소스를 캐시합니다.
3. 이후 인터넷이 없어도 최근 캐시 기준으로 앱이 열립니다.
4. 체중 기록과 체크 상태는 아이폰 브라우저 저장소에 남습니다.
